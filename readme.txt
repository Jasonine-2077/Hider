Hider - March 2021
==========================

Hider，一键轻松隐藏你的程序。
本软件基本原则为简单易用。
所有使用方式都可用快捷键开启。
具体用法参见下文。

==========================

直接下载：(https://codeload.github.com/kingerlcy/Hider/zip/main/)

==============================================================

快捷键
=============================================
 按键                    用法
 ---------------------- ---------------------
 Ctrl+H                 隐藏当前程序
 Ctrl+Shift + H         显示最后一个隐藏的程序
 Shift+Table + H        显示所有隐藏的程序
 Ctrl+Q                 退出Hider
 Shift+P                暂停Hider
 Ctrl+K                 退出当前程序
