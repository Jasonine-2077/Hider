Hider - January 2021
==========================

Get the latest version [from the Github website](https://https://github.com/kingerlcy/Hider).

===========================================================================

Controls
===========================================================================

| Key                     | Action                                                          |
| ----------------------- | --------------------------------------------------------------- |
| Ctrl + H                | Hide your program                                               |
| Ctrl + Shift + H        | Unhide your program                                             |
| Shift + Table + H       | Unhide all programs                                             |
| Ctrl + Q                | Quit                                                            |
| Shift + P               | Pause Hider                                                     |
===========================================================================
